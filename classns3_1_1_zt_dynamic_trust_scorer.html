Detailed Description

Implements dynamic trust evaluation and enforcement.

The trust scorer maintains trust values for communication sessions identified by source and destination node pairs.

Trust is updated according to observed packet behavior. Repeated violations result in exponential trust decay, while normal communication enables gradual recovery.

Sessions falling below the trust threshold are blocked.
