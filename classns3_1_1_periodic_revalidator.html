Detailed Description

Continuously revalidates trust of active nodes.

CLASS: PeriodicRevalidator

This class periodically evaluates contextual drift, updates trust scores, and triggers policy enforcement actions when trust falls below a configured threshold.

It implements the Zero Trust principle of continuous verification after initial authentication.

◆ Revalidate()

void ns3::PeriodicRevalidator::Revalidate ( )

Performs periodic trust revalidation.

For each active node:

  • Computes contextual drift.
  • Updates trust score.
  • Evaluates trust threshold compliance.
  • Revokes access if trust falls below the threshold.

This function implements continuous verification and adaptive access control within the Zero Trust framework.
